# YYY Network Switcher

基于 Python 开发的 Windows 双网卡路由切换工具。通过动态调整网络接口跃点数 (Metric)，实现低延迟游戏（如走 Wi-Fi 手机热点）与大带宽下载（如走有线宽带）的物理级网络分流。

## 核心特性

- **原生系统级分流**：调用 Windows 底层 `netsh` 命令修改路由表优先级，零进程注入，完全兼容 Vanguard、EAC 等内核级反作弊系统。
- **无缝网络回落**：基于跃点数调度机制，当高优先级网络（如热点）断开时，流量会自动回落至备用网络，避免彻底断网。
- **现代 UI 与便捷悬浮挂件**：主控制台采用暗黑圆角卡片式设计，视觉精致、数据直观；同时配备桌面悬浮窗，完美兼顾界面美感与全局快捷切换体验。
- **真实流量监控**：集成 `psutil` 读取系统底层网卡物理收发数据，直观验证分流状态。
- **极低资源开销**：采用事件驱动与对象互调架构同步 UI 状态，支持系统托盘静默常驻。

## 环境依赖

- **操作系统**: Windows 10 / 11

- **运行环境**: Python 3.8+

- **依赖库**:

  ```
  pip install customtkinter psutil pystray pillow
  ```

## 使用说明

1. **部署运行**：
   - 将源码文件保存为 `NetSwitcher.pyw`（`.pyw` 后缀可屏蔽控制台黑框，实现纯 UI 后台运行）。
   - 双击运行，因涉及系统网络配置修改，请在 UAC 弹窗中允许管理员权限。
2. **功能配置**：
   - 在主控制台下拉菜单中指定当前连接的“无线网络”与“有线网络”接口。
   - **游戏模式**：将 Wi-Fi 跃点数提至最高（Metric=10），新建连接将优先通过低延迟热点。
   - **普通模式**：将有线网络跃点数提至最高，恢复常规大带宽网络环境。
3. **注意事项**：
   - 路由规则仅对切换后新建的网络连接（Socket）生效。对于长连接应用（如网游客户端），请先切换网络模式，再启动应用程序。对于短连接应用（如浏览器），刷新即可生效。
